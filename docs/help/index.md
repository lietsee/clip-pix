# ClipPix ヘルプ

## ClipPixとは

ClipPixは、クリップボードにコピーした画像やテキストを自動的に保存し、Pinterest風のグリッド表示で管理できるデスクトップアプリケーションです。

### 主な機能

- **クリップボード監視**: 画像やテキストをコピーするだけで自動保存
- **URL画像ダウンロード**: 画像URLをコピーすると自動でダウンロード
- **グリッド表示**: カードを自由にリサイズ・並び替え
- **プレビュー**: ダブルクリックで拡大表示
- **お気に入り・メモ**: 画像に3段階のお気に入りとメモを追加

### 対応OS

- Windows 10/11
- macOS 12以降
