# カードの操作方法

ClipPixでは画像・テキスト・PDFがそれぞれカードとして表示されます。このページでは各カードの操作方法を説明します。

---

## 画像カード

### 基本操作

| 操作 | 方法 |
|------|------|
| **選択** | カードをクリック |
| **プレビュー** | ダブルクリック または `Enter` キー |
| **コピー** | `Ctrl+C`（選択中のカード） |

### リサイズ

カードにマウスを乗せると、四隅に**リサイズハンドル**が表示されます。

- **ハンドルをドラッグ**: カードのサイズを変更
- **列幅にスナップ**: 自動的に列幅の倍数に揃います
- **サイズリセット**: `Shift+Enter` で200×200にリセット

### ズームとパン

画像カード内で画像を拡大・縮小・移動できます。

| 操作 | 方法 |
|------|------|
| **ズームイン** | `Ctrl+=` または `Ctrl++` |
| **ズームアウト** | `Ctrl+-` |
| **マウスホイールズーム** | 右クリックを押しながらホイール回転 |
| **パン（移動）** | 右クリックを押しながらドラッグ |

> ズーム範囲: 0.5倍 ～ 15倍

### 並び替え

カード下部中央の**ドラッグハンドル**（横線3本）をドラッグすると、カードの順序を変更できます。

- ドラッグ中は他のカードが自動的に移動
- 画面端に近づくと自動スクロール

### お気に入り

カード左下の**ハートボタン**でお気に入りレベルを設定できます。

| レベル | 色 | 意味 |
|--------|-----|------|
| 0 | なし | 未設定 |
| 1 | 緑 | 低 |
| 2 | オレンジ | 中 |
| 3 | ピンク | 高 |

クリックするたびにレベルが上がり、3の次は0に戻ります。

### メモ

カード左上の**メモボタン**（ホバーで表示）をクリックすると、メモを追加・編集できます。

- メモがあるカードにホバーすると、0.5秒後にツールチップでメモ内容が表示されます

### 削除

カード右上の**削除ボタン**（ゴミ箱アイコン）をクリックすると、確認後にゴミ箱（`.trash`フォルダ）に移動します。

---

## テキストカード

テキストファイル（`.txt`）はテキストカードとして表示されます。

### 基本操作

| 操作 | 方法 |
|------|------|
| **プレビュー** | ダブルクリック または `Enter` キー |
| **コピー** | `Ctrl+C`（テキスト内容をコピー） |

### インライン編集

カード内のテキストをダブルクリックすると、その場で編集できます。

| 操作 | キー |
|------|------|
| **保存** | `Ctrl+S` |
| **キャンセル** | `Escape` |

### リサイズ

画像カードと同様に、四隅のハンドルでリサイズできます。

---

## PDFカード

PDFファイルはサムネイル付きのカードとして表示されます。

### 基本操作

| 操作 | 方法 |
|------|------|
| **プレビュー** | ダブルクリック または `Enter` キー |

### プレビューウィンドウでの操作

PDFプレビューウィンドウでは以下の操作が可能です：

| 操作 | キー |
|------|------|
| **前のページ** | `←` (左矢印) |
| **次のページ** | `→` (右矢印) |
| **最初のページ** | `Home` |
| **最後のページ** | `End` |
| **フルスクリーン** | `F11` |
| **閉じる** | `Escape` または `Ctrl+W` |

---

## プレビューウィンドウ

ダブルクリックまたは `Enter` キーで開くプレビューウィンドウは、別ウィンドウとして表示されます。

### 共通操作

| 操作 | キー |
|------|------|
| **閉じる** | `Escape` または `Ctrl+W` |
| **常に最前面** | ウィンドウ右上のピンボタン |

### 画像プレビュー

- マウスホイールでズーム
- ドラッグで画像移動
- `Ctrl+C` でクリップボードにコピー

---

## ショートカットキー一覧

### グリッド画面

| キー | 操作 |
|------|------|
| `Ctrl+M` | ミニマップ表示切替 |
| `Enter` | 選択カードをプレビュー |
| `Shift+Enter` | カードサイズをリセット |
| `Ctrl+C` | 選択カードをコピー |
| `Ctrl+=` / `Ctrl++` | ズームイン |
| `Ctrl+-` | ズームアウト |

### テキストカード編集中

| キー | 操作 |
|------|------|
| `Ctrl+S` | 保存 |
| `Escape` | キャンセル |

### PDFプレビュー

| キー | 操作 |
|------|------|
| `←` / `→` | ページ移動 |
| `Home` / `End` | 最初/最後のページ |
| `F11` | フルスクリーン |
| `Escape` / `Ctrl+W` | 閉じる |
